Google OAuth 설정
각자 자신의 Google Cloud 프로젝트 Client ID를 사용합니다.
내 구독 데이터는 브라우저 안에서만 처리되며 외부 서버로 전송되지 않습니다.
Google Cloud Console → 사용자 인증 정보 → OAuth 2.0 클라이언트 ID
처음 사용 시 설정 방법 (1회만)
1. Google Cloud Console → 새 프로젝트 생성
2. API 및 서비스 → 라이브러리 → YouTube Data API v3 사용 설정
3. OAuth 동의 화면 → 외부 선택 → 앱 이름/이메일 입력
4. 사용자 인증 정보 → + 만들기 → OAuth 2.0 클라이언트 ID → 웹 애플리케이션
5. 승인된 JavaScript 원본에 이 페이지 URL 추가
6. 생성된 클라이언트 ID를 위 입력란에 붙여넣기
